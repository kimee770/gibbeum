# GIBBEUM 프로젝트 스타일 규칙

## 텍스트 정렬
- 모든 본문(paragraph) 텍스트는 **왼쪽 정렬(left-align)**을 기본으로 한다.
- `text-justify`, `text-right`는 본문에 사용하지 않는다.
- `text-center`는 UI 컴포넌트(카드 내부 수치, 테이블 헤더, 섹션 제목 등) 에만 허용한다.
